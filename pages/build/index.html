通用 URL 代理（根路径）

URL 参数
  • 代理路径: /
  • 使用示例: 
    https://cf.liuass.eu.org/raw.githubusercontent.com/ACL4SSR/ACL4SSR/refs/heads/master/Clash/config/ACL4SSR_Online.ini
重要提示：
  • URL 参数模式: 代理路径后必须跟上完整的、在白名单内的 URL。
  • 路径映射模式: 代理路径后直接跟上目标站点的文件路径。
  • Host 模式 (透明转发): 专为需要保留 Host 头等信息的应用设计。它会剥离代理路径，将剩余路径完整转发到目标域名（当然也可以像我一样用来来缩短域名）。
